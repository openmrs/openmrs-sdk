OpenMRS SDK Version 1.0

Description:
OpenMRS-SDK allows the user to create, develop and manage a module they are working on.
The SDK includes a variety of features and commands such as:

omrs-run (runs an embedded instance of OpenMRS)
omrs-create-project (creates a project which will allow the user to
					develop multiple modules and deploy them to an embedded instance of OpenMRS)
omrs-create-module (creates a basic module)
omrs-version (displays the current version of the SDK including java and maven info)
omrs-help (displays all available commands)
